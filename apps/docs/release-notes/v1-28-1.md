# 1.28.1

## Added

- **Markdown rendering now supports the expanded Nizel plugin set.** Girk and the SDK now enable first-party Nizel plugins for abbreviations, autolinks, citations, code copy buttons, definition lists, details blocks, diagrams, emoji, footnotes, frontmatter UI blocks, heading anchors, math wrappers, media figures, output sanitizing, Shiki highlighting, tables of contents, and typography extensions.

## Changed

- **Syntax highlighting now uses the latest compatible Nizel and Shiki packages.** The SDK now uses Shiki 4 for compatibility with the latest `nizel-plugin-shiki`.

## Fixed

- **Inter-page Markdown links now resolve correctly.** Links written with a `.md` extension (e.g. `[Setup](docs/setup-new-project.md)`, the format used by GitHub) are now rewritten to Girk's URL format during the build, so they work on the generated site instead of pointing to missing `.md` routes.
