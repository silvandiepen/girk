# 1.11.0

Released on 2026-03-23.

## Changed

- The `Styling` feature page is now `Customisation`, which better matches that the page covers CSS hooks, script loading, web components, and Vue-powered enhancements.
- Archive cards now use larger icons, no card borders, and a cleaner elevated treatment.
- The theme switcher is now a real light/dark toggle, and project logos now appear in the footer when available.
- Header and footer navigation now use more compact background-based hover and active states with tighter radii and clearer submenu interactions.
- Header subnavigation now opens under the owning item instead of spanning the full header, with tighter spacing and proper hover and active states for submenu links.

## Fixed

- The homepage link is no longer incorrectly treated as active or as the parent of every route in the header navigation.
