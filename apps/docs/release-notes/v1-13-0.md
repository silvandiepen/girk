# 1.13.0

Queued from the current local `main` branch and intended to release next.

## Changed

- `example-blog` now includes fuller editorial sample posts so the archive reads like a real blog instead of a placeholder fixture.
- `example-recipes` now reads like a small cookbook, with category pages, many more classic dishes, and illustrative recipe images.

## Added

- Girk can now load JSON from a remote endpoint or local file at build time, use `{{result...}}` placeholders in frontmatter and Markdown, repeat blocks with `{{#each result}}`, and generate many detail pages from one Markdown template with `dataSlug`.
- A dedicated `example-data` project now demonstrates build-time data sources without overloading `example-basic`.
