# 1.30.0

## Added

- **Math and diagram pages now load their renderers automatically.** Pages with math load KaTeX, pages with Mermaid diagrams load Mermaid, and pages without those features avoid the extra assets.

## Changed

- **Code blocks and heading anchors have more polished default styling.** Copy buttons now sit inside styled code containers, and heading links appear as subtle hover/focus affordances instead of visible trailing hash marks.
