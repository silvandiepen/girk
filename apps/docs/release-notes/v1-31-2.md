# 1.31.2

Released on 2026-07-03.

## Fixed

- Restored styling for the latest Nizel plugin output, including code copy blocks.
- Fixed code copy buttons so they copy from Nizel's current hidden source field and still fall back when the Clipboard API is unavailable.
- Enabled the remaining first-party Nizel plugin syntax in Girk, including badges, keyboard shortcuts, Open Icon icons, print controls, and task lists.
- Rendered Markdown task lists now use Nizel's task-list plugin in editable mode, so checkboxes can be toggled in the browser.
