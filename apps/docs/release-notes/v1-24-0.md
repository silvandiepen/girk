# 1.24.0

Queued from the current local `main` branch.

## Added

- `Search`: Projects can now enable a fully static local search index with `projectSearch`, including generated search shards, a built-in browser client, and page-level overrides.

## Changed

- `Docs`: Feature guides can now declare accent colors in their page metadata so archive cards get stronger visual distinction.
- `Docs`: Added a dedicated Search feature guide with setup, markup, and tuning examples.
- `Docs`: Combined the separate page settings and page metadata docs into one fuller page frontmatter reference, including defaults, key behavior, and usage patterns.
- `Navigation`: Hidden pages no longer surface in generated archive and tag listings, which keeps compatibility pages and helper content out of surfaced docs UI.
- `Theme`: Content links now use the page accent styling, and linked inline code gets a clearer visual cue.
