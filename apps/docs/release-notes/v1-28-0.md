# 1.28.0

Released on 2026-05-04.

## Changed

- **Replaced markdown-it with Nizel** across both `girk-sdk` and `girky`. Nizel is a custom CommonMark-compliant parser with no Node.js-specific dependencies, making the full pipeline compatible with Cloudflare Workers and browser environments.
- **Replaced Pug templates with EJS** across both packages. Templates are now compiled as inline strings — no filesystem access required at render time.
- **Replaced `node-fetch`** with native `fetch` (available in Node 18+ and all modern runtimes).
- **Removed `date-fns`** in favor of `Intl.DateTimeFormat` for date formatting.

## Removed

- `markdown-it` and 8 plugins (anchor, alert, deflist, emoji, highlightjs, prism, tasks)
- `pug` and all Pug template files
- `node-fetch`, `date-fns`, `svgo`, `purgecss`, `purify-css`
- `@types/markdown-it`, `@types/pug`, `@types/svgo`
- Custom `markdown-it-article` and `markdown-it-svg` plugins

## Added

- `nizel`, `nizel-plugin-alert`, `nizel-plugin-deflist`, `nizel-plugin-emoji`, `nizel-plugin-shiki`
- Self-contained EJS compiler in both `girk-sdk` and `girky`
- `girk-sdk` is now an ESM module (`type: module`)
