# 1.17.0

Queued from the current local `main` branch and intended to release next.

## Added

- Pages and section archive parents can now define `color`, `backgroundColor`, and `textColor` in frontmatter to map into section-scoped CSS variables for more intentional local theming.

## Fixed

- First-level section landing pages such as `icons/README.md` and `packages/README.md` now appear in the generated top navigation even when they do not define `archive` metadata.
