# 1.12.0

Queued from the current local `main` branch and intended to release next.

## Changed

- The root repository `README.md` now explains Girk as a product first, with quick-start guidance and core generator concepts ahead of the monorepo details.
- The npm package `README.md` now includes product-level guidance, installation, quick-start usage, config hooks, and links to the full docs and examples.
- Header subnavigation panels now size to their content more tightly instead of keeping a wide desktop flyout.
- Footer navigation groups now size to their content, the footer switches share the same segmented styling, and the footer stays pinned through the page shell layout instead of a viewport-height page container.
- `example-multilang` now stays focused on multilingual routing, with a plain example title and the default shared theme instead of extra styling assets or branding.
- `example-blog` now includes fuller editorial sample posts so the archive reads like a real blog instead of a placeholder fixture.
