# Examples

These are complete Girk projects you can inspect, run, and compare against their live output.

## What This Section Is For

Use these examples when you want to:

- see what a specific Girk feature looks like in a finished site
- understand which content and settings produced that result
- copy a working starting point into your own project
- compare different project shapes before you commit to one

Each example section below includes:

- the live deployed site
- the source folder in GitHub
- a short explanation of what the example is proving
